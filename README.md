# ASUS/ROG Secure Boot 2023 Assistant

<a href="./README_EN.md">English</a>

给 ASUS / ROG Windows 设备用的 Secure Boot 2023 证书检测与修复助手

## 基本要求

- 支持安全启动的 Windows 系统
- ASUS、ROG 或 ASUSTeK 设备
- 管理员权限

*非 ASUS/ROG 设备只开放只读检测*

## 使用前

- 备份重要文件。
- 关闭 BitLocker / 设备加密。
- 请参考 [教程](https://www.asus.com/tw/support/faq/1049829/) 开启 Setup Mode，并清空 Secure Boot Keys。
- 不要随意使用 Restore Factory Keys。 *部分 BIOS 默认 Keys 不包含完整 2023 证书，恢复后可能回到旧证书*
