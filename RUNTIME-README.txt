ASUS/ROG Secure Boot 2023 Assistant v1.0.1.2

System requirements:
- Windows 10 or Windows 11
- 64-bit Windows (x64)
- UEFI boot mode
- ASUS, ROG, or ASUSTeK device for controlled firmware writes
- Administrator privileges

Start:
1. Extract the complete ZIP.
2. Keep ASUS-ROG-SecureBoot-2023-Assistant.exe.config beside the EXE. It enables long-path support.
3. Run ASUS-ROG-SecureBoot-2023-Assistant.exe.
4. Read the first-run safety notice and follow only the highlighted next action.

Important:
- The executable is not Authenticode-signed. Verify the package hashes before use.
- Do not upload repair workflow recovery packages, raw Default Keys backups, BitLocker recovery keys, serial numbers, or personal files.
- PK is always written last. Restart resume performs detection only and never starts a new UEFI write automatically.
- Non-ASUS/ROG devices are read-only.
